Sparkker - Stock Analysis with Spark

Learning a bit of AngularJS and Spark, so this prototype is used for graphically display what is calculated in the back-end.

  • You have more information about the project in the wiki along with instructions on how to lunch all the services.

Technology stack

  • AngularJS in the front-end.
  • Angular Chart to show stock charts.
  • Spring Boot 1.3.3 in the Server Side.
  • Drools 6.4 as rule engine
  • Spark running in a single node to do the calculations

The source code is available in this GitHub repository.

  • Feel free to fork it, however I am still cleaning up things (like the commit policy, there is neither CI nor automatic testing - for the moment)
  • My objective is to turn this sandbox repository in something closer to an open-source project
